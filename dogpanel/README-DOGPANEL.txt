DogPanel — админ-панель для сайта DogStore
======================================

Расположение: папка dogpanel/

Функции:
* Авторизация admin / ilovedogs
* Управление собаками (добавление, редактирование, удаление, загрузка фото)
* Управление новостями (CRUD + изображение)
* Модерация отзывов (одобрить/скрыть, удалить)

База данных:
* см. sql/dogpanel_schema.sql
* имя БД по умолчанию — dogpanel (настраивается в config.php)
